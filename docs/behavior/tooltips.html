tooltips

  1. - behavior in module wk.chart

enables the display of tooltips. See the tooltips section in the guide for more details

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <chart
  [tooltips=""]>
...
</chart>

Arguments

Param Type Description
tooltips
(Accepted Values: true, false, path/to/custom-template.html)
 booleanurl

enable / disable tooltips, resp. supply a custom tooltip template url. If no template url is supplied, a (configurable) default template is used (see here for how to configure the default template),
