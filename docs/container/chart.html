chart

  1. - container in module wk.chart

chart is the container directive for all charts.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <chart
  data=""
  [deep-watch=""]
  [filter=""]
  [title=""]
  [subtitle=""]
  [animation-duration=""]>
...
</chart>

Arguments

Param Type Description
data
array

Data to be graphed, ...more
deep-watch
(optional, default: false)
 boolean
filter
(optional
