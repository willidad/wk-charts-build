chart

  1. - container in module wk.chart

chart is the container directive for all charts.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <chart
  data=""
  [deep-watch=""]
  [filter=""]
  [header=""]
  [header-style=""]
  [sub-header=""]
  [sub-header-style=""]
  [edit=""]
  [edit-selection=""]
  [animation-duration=""]>
...
</chart>
