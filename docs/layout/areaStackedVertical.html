Layout areaStackedVertical

draws a area chart layout

Usage

<layout area-stacked-vertical [area-stacked-vertical=""] [markers=""] [spline=""] [area-style=""] ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
x
type = linear
domainRange = total
y
type = linear
domainRange = extent
color
type = category20

Arguments

Param Type Description
areaStackedVertical
(optional, default: zero)
(Accepted Values: zero, silhouette, expand, wiggle)
 string

Defines how the areas are stacked. For a description of the stacking algorithms please see d3 Documentation on Stack Layout
markers
(optional, default: false)
(Accepted Values: true, false)
 boolean

show a data maker icon for each data point
spline
(optional, default: false)
(Accepted Values: true, false)
 boolean

interpolate the area shape using bSpline
areaStyle
(optional
 object

Set the pie style for columns lines in the layout
