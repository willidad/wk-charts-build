Layout columnHistogram

draws a histogram chart layout

Usage

<layout column-histogram [labels=""] ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
rangeX
type = linear
domainRange = extent
y
type = linear
domainRange = extent
color
type = category20

Arguments

Param Type Description
labels
(optional, default: true)
(Accepted Values: true, false)
 boolean

controls the display of data labels for each of the bars.
