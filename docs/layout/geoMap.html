Layout geoMap

Draws a map form the `geoJson` data provided. Colors the map elements according to the data provided in the cart data and the mapping rules provided in the `idMap` attribute. The map is drawn according to the properties provided in the `projection` attribute For a more detailed description of the various attributes, and a reference to geoJson, projections and other relevant topic please see the geoMap section in the guide

Usage

<layout geo-map projection="" geojson="" ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
color
type = category20

Arguments

Param Type Description
projection
object

sets the projection attributes for the map defined in geojson
geojson
object

the geojson object that describes the the map.
