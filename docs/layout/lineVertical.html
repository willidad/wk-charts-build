Layout lineVertical

Draws a vertical line chart layout

Usage

<layout line-vertical [markers=""] [spline=""] [line-style=""] ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
x
type = linear
domainRange = extent
y
type = linear
domainRange = extent
color
type = category20

Arguments

Param Type Description
markers
(optional, default: false)
(Accepted Values: true, false)
 boolean

show a data maker icon for each data point
spline
(optional, default: false)
(Accepted Values: true, false)
 boolean

interpolate the line using bSpline
lineStyle
(optional
 object

Set the line style for all lines in the layout
