Layout lineVertical

Draws a vertical line chart layout

Usage

<layout line-vertical [markers=""] ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
x
type = linear
domainRange = extent
y
type = linear
domainRange = extent
color

Param Type Description
