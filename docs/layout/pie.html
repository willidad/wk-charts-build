Layout pie

draws a pie chart layout

Usage

<layout pie [labels=""] [label-style=""] ... >
  ...
</layout>
Required DimensionsDimension Attribute Defaults
size
type = linear
domainRange = extent
color
type = category20

Arguments

Param Type Description
labels
(optional, default: true)
(Accepted Values: true, false)
 boolean

controls the display of data labels for each of the pie segments.
labelStyle
(optional, default: font-size:"1.3em")
 object

defined the font style attributes for the labels.
